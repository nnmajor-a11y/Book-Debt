At A Glance:

How To Understand Your Book Debt Sheet
Terms Meaning Example
Negative 1
in the "Evaluation Integer" Cell.
  • The User has bought a book.
  • Has NOT READ the book listed in the row.
  • Debt Earned
 -1
Zero
in the "Evaluation Integer" Cell
  • The User has the book listed in the row.
  • Canceled-Out Debt for One book.
 0
Positive Numbers
in the "Evaluation Integer" Cell.
'x' being any number.
  • The number of times the book listed in this row HAS been READ.
  • Credit Earned
 + X

i.e., "The user read this bought book 'x' many times."
Any Negative Number Below (-1)
in the "Evaluation Integer" Cell.
'-x'
  • The number of times the book listed in this row has been bought.
  • Ranked up debt with the same book
 - X

i.e., "The user bought the same book 'x' many times."
Any Positive Number
in the "Evaluate Next Purchase" Cell.
  • You CAN BUY more books!
Accumulated Credit

Buy the number of books according to this Positive Number Shown.
Any Negative Number
in the "Evaluate Next Purchase" Cell.
  • You CAN'T buy any books.
Accumulated Debt

Read the number of books According to this Negative Number Shown.
